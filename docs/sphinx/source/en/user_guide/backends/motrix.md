# Motrix Backend

`motrixsim-core` provides multithreaded CPU stepping plus native snapshot/playback.

- Asset format: MJCF subset.
- Headless video export works out of the box on macOS and Linux.
- Snapshot/restore capability is required by some task owners — see capability declaration in YAML.

See {py:mod}`unilab.base.backend.motrix` for the adapter API.
