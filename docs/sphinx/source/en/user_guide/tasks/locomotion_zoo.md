# Locomotion Zoo

UniLab ships locomotion tasks for:

- **Unitree G1** humanoid — whole-body motion tracking + flips.
- **Unitree Go1 / Go2** quadrupeds — joystick (flat & rough).
- **Go2W** wheel-legged variant.
- **Go2 + Airbot arm** — combined locomotion + manipulation.

Browse the per-robot configs under `conf/ppo/<task>/<backend>.yaml` and the env modules under `src/unilab/envs/locomotion/`.
