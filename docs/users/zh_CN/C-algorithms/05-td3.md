# TD3

语言: 简体中文

TD3 和 SAC 共用同一套 off-policy 训练入口，但 owner 路径和超参数不同。

## 默认入口

```bash
uv run train --algo td3 --task g1_walk_flat --sim mujoco
```

## 适用场景

- off-policy 训练
- 与 SAC 共享同一脚本入口，但 owner 路径不同

## 运行模型

- 和 SAC 共用 shared-memory off-policy runner
- 常见用途是用同一任务 owner 做 SAC / TD3 对照
- 是否同步采集由 `training.no_sync_collection` 控制

## 关联脚本

```bash
uv run scripts/train_offpolicy.py algo=td3 task=td3/g1_walk_flat/mujoco
```

## 常见命令

```bash
uv run train --algo td3 --task g1_walk_flat --sim mujoco

uv run scripts/train_offpolicy.py \
  algo=td3 \
  task=td3/g1_walk_flat/mujoco \
  training.no_play=true

uv run scripts/train_offpolicy.py \
  algo=td3 \
  task=td3/g1_walk_flat/mujoco \
  training.play_only=true \
  algo.load_run="2024-02-04_12-00-00"
```

## 什么时候优先看 TD3

- 任务 owner 已经为 TD3 单独调过超参数
- 你需要和 SAC 做同任务对照
- 你想继续沿用同一套 off-policy 训练栈，但换成 TD3 owner

## 日志根目录

```text
logs/fast_td3/<task>/
```
